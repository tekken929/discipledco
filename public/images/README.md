# Images Folder

This folder contains all local images for the Discipled Co. website.

## Logo Files

- `logo.svg` - Full circular logo with dark blue background (for dark mode/general use)
- `logo-light.svg` - Full circular logo with white background (for light mode)
- `logo-icon.svg` - Simplified square icon version (for favicon, app icons, small displays)

## Usage

All images can be referenced using relative paths:

```jsx
<img src="/images/logo.svg" alt="Discipled Co. Logo" />
```

Or in Vite/React:

```jsx
<img src="/images/logo.svg" alt="Discipled Co. Logo" />
```

## Adding New Images

Place all static images in this folder and reference them using `/images/filename.ext`.
